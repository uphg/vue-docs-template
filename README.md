## Vue 3 官方文档 Vuepress 模板

从 Vue 3 官方文档中拷贝的源文件，删去了不必要的配置。

官方地址：[https://github.com/vuejs/docs-next](https://github.com/vuejs/docs-next)

## 开发

**1. 拷贝仓库**

```sh
git clone git@github.com:uphg/vue-docs-template.git
```

**2. 初始化**

```sh
yarn # or npm install
```

**3. 运行测试**

```sh
yarn serve # or npm run serve
```
